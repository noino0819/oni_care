# Supabase 이메일 확인 설정 변경 가이드

브라우저 자동화에 일시적인 문제가 발생하여 직접 설정을 변경해야 합니다.

## 단계별 안내

1. **Supabase Dashboard에서 프로젝트 선택**
   - 현재 열려있는 Supabase 탭 사용
   - 프로젝트 리스트에서 프로젝트 클릭

2. **Authentication 메뉴로 이동**
   - 좌측 사이드바에서 **Authentication** 클릭

3. **Providers 탭 선택**
   - 상단 탭에서 **Providers** 클릭

4. **Email Provider 설정 열기**
   - Provider 목록에서 **Email** 클릭

5. **이메일 확인 비활성화**
   - **"Confirm email"** 토글을 **OFF**로 변경
   - (이메일 확인을 요구하지 않도록 설정)

6. **저장**
   - 하단의 **Save** 버튼 클릭

## 완료 후

설정 변경을 완료하시면 알려주세요. 
회원가입 기능을 다시 테스트하겠습니다.
